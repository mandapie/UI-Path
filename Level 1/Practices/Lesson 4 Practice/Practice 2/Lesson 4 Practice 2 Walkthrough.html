Practical Exercise - Walkthrough

First, add an Input Dialog activity and fill it in with the proper parameters.

  • Find and add an Input Dialog activity to the Main panel.

    • Enter ‘Get the Temperature’ as the title (remember to surround it in quotation marks).

    • Enter ‘Enter a city name’ as the label.

    • Create a string or generic variable called city and enter it as the output.

The recorder can be used to simplify the process of creating the main part of the sequence.

  • Open a browser and navigate to google.com (Internet Explorer recommended)

  • In the Design tab, select Recording > Web. The Web Recording toolbar is displayed.

  • Click Open Browser and indicate the opened browser.

    • Make sure to exclude the browser window bar at the top.

    • The recorder should prompt you to confirm the URL.

  • Click Type and select the Google search bar.

    • Type ‘weather in new york’ and press the Enter key to record it.

  • From the Type menu, select Send Hotkey.

    • Select the browser area again (excluding the top window bar).

    • From the dropdown, select enter.

  • From the Text menu, select Scrape> Screen Scraping.

    • Using the element selector, click the temperature numbers.

  • From the Open Browser menu, select Close Browser and indicate the browser area to close.

  • Click Save & Exit

The final part involves modifying the input of the search to match the user’s city, the outputs of the scraped temperature, as well as adding a message box to display the temperature.

  • Navigate to the Type Into inside the Do container.

    • Replace “weather in new york” with "’“weather in “ + city"’.

  • Navigate to the Get Full Text activity below the Send Hotkey one.

    • Create a generic or string variable called temperature and use it as the output parameter.

  • Add a Message Box activity to the end of the workflow.

    • For the text parameter, type: “The weather in “ + city + “ is: “ + temperature + “ degrees”.

      • This concatenates user-friendly sentence fragments with the corresponding values that were extracted as variables.
