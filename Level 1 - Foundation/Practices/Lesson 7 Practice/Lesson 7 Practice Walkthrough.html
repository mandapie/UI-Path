Practical Exercise - Walkthrough

This may seem like a tough exercise due to the constraints, but luckily, Studio's Citrix Recorder makes it very simple.

  • Download and open the ExpenseIt executable.
  • On the Design tab navigate to the Recording button on the navigation menu and select Citrix Recorder.

Filling in form fields like those on ExpenseIt can be done with image or text recognition. For the first field, let's use Click Text.

  • Select Click Text and select a box around the Email: label.

  • When the Screen Scraping Wizard appears, navigate to the Search Text field and type in Email.
  • Click Set mouse position and then Indicate a point and click inside the text box next to the email label.
  • Set the OCR engine to Microsoft OCR and click Finish.
  • On the recorder menu, select Type.
  • Use the entire ExpenseIt window as the element selection.
  • Type in: training@uipath.com
  • Check the Empty field, and hit enter

For Cost Center, use image recognition just like you did for employee number click in the dropdown menu and expand it.

  • Next, select Click Text and use the entire window as the selection element (simulating a citrix environment).
  • In the Search Text field, type: Marketing.
  • Set the OCR engine to Google for this one.
  • For Employees, select Click Text again.
  • Use the entire window as the selection area again.
  • In the Text to be found field, type: CSG.
  • Use Microsoft OCR for this one.
  • Do the same steps for selecting an employee name, this time selecting the text: Amy.
  • Select Click Image and select an area around the text on the Create Expense Report button
  • Click Save & Exit and verify that all Microsoft and Google OCR engines used use the Legacy profile
