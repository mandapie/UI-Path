Practical Exercise - Walkthrough

The most direct way of designing a workflow which tries finding all the 3-4 possible images at the same time is by using a Pickblock with various PickBranch activities.
  1. Drag a Pick activity / container to the Main panel.
  2. Drag three Pick Branch activities inside the Pick container, one next to each other.
  3. Take the first Pick activity, and in the Trigger area, drag an Image Exists activity.
    • Click the Indicate Image on screen button and use the mouse to click and drag to surround the ExpenseIT icon with a rectangle in the Remote Desktop Window.
  4. In the Action area of the Pick activity, drag a Click Image activity.
    • Click the Indicate Image on screen button and use the mouse to click and drag to surround the ExpenseIT icon with a rectangle in the Remote Desktop Window.
    • Go to the ClickType property and change it to CLICK_DOUBLE.
  5. Go to the Remote Desktop Connection window and select the icon so that it gets a transparent highlighting square around it.
  6. Go the the second Pick Branch activity and repeat steps 3 & 4 for the new state of the icon.
  7. Go to the Remote Desktop Connection and change the screen resolution.
  8. Go to the third Pick Branch activity and repeat steps 3 & 4 for the new appearance of the icon, due to the resolution change.


This way, whichever of the 3 images is found on the screen, the Click Image will actually apply on the actual one.
